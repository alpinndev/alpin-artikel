Pengenalan Bahasa Pemrograman Web dan Data

Dalam dunia teknologi informasi, bahasa pemrograman memegang peran penting dalam pengembangan aplikasi web dan pengolahan data.

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif.

Dasar-dasar JavaScript:

  • Variabel: Digunakan untuk menyimpan nilai.
  • Tipe data: JavaScript memiliki beberapa tipe data, seperti number, string, boolean, dan object.
  • Operator: Digunakan untuk melakukan operasi, seperti penjumlahan, pengurangan, dan perbandingan.
  • Kontrol flow: Digunakan untuk mengontrol alur program, seperti if-else, switch, dan loop.
  • Fungsi: Digunakan untuk membuat kode yang dapat digunakan kembali.

Contoh kode JavaScript untuk menampilkan pesan

alert("Halo, Selamat Datang!");

Contoh kode JavaScript untuk mengubah warna teks

document.getElementById("teks").style.color = "blue";

HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur dan konten halaman web.

Dasar-dasar HTML:

  • Tag: Digunakan untuk membuat elemen HTML, seperti paragraf, judul, dan link.
  • Atribut: Digunakan untuk memberikan informasi tambahan tentang elemen HTML.
  • Elemen: Digunakan untuk membuat konten halaman web, seperti teks, gambar, dan video.

CSS

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan layout halaman web.

Dasar-dasar CSS:

  • Selektor: Digunakan untuk memilih elemen HTML yang ingin diberi style.
  • Properti: Digunakan untuk memberikan nilai pada selektor.
  • Nilai: Digunakan untuk memberikan nilai pada properti.

Contoh kode CSS untuk mengatur warna latar belakang

body { background-color: #f2f2f2; }

Contoh kode CSS untuk mengatur ukuran font

h1 { font-size: 36px; }

SQL

SQL (Structured Query Language) adalah bahasa pemrograman yang digunakan untuk mengelola dan mengolah data dalam database.

Dasar-dasar SQL:

  • SQL

    SQL (Structured Query Language) adalah bahasa pemrograman yang digunakan untuk mengelola dan mengolah data dalam database.

    Dasar-dasar SQL:

    • Perintah: Digunakan untuk melakukan operasi pada database, seperti SELECT, INSERT, UPDATE, dan DELETE.
    • Klausa: Digunakan untuk memfilter data, seperti WHERE dan GROUP BY.
    • Fungsi: Digunakan untuk melakukan operasi pada data, seperti SUM dan COUNT.

    Contoh kode SQL untuk membuat tabel

    nama_tabel(tipe_kolom1,tipe_kolom2,tipe_kolom-N).

    Contoh kode SQL untuk menambahkan data

    VALUES (1, 'John Doe', 'Jalan Raya No. 1');

    Python

    Python adalah bahasa pemrograman yang digunakan untuk pengembangan aplikasi, pengolahan data, dan lain-lain.

    Dasar-dasar Python:

    • Variabel: Digunakan untuk menyimpan nilai.
    • Tipe data: Python memiliki beberapa tipe data, seperti integer, float, string, dan list.
    • Operator: Digunakan untuk melakukan operasi, seperti penjumlahan, pengurangan, dan perbandingan.
    • Kontrol flow: Digunakan untuk mengontrol alur program, seperti if-else, switch, dan loop.
    • Fungsi: Digunakan untuk membuat kode yang dapat digunakan kembali.

    Contoh kode Python untuk menampilkan pesan

    print("Halo, Selamat Datang!")
